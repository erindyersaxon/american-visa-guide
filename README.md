# AVG MCP Server

A minimal Model Context Protocol server scaffold using Node.js and TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start in development mode:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Run the built server:
   ```bash
   npm start
   ```

## Endpoints

- `GET /` - Basic server status
- `GET /health` - Health check
