  Vercel detects changes

    ↓

    ↓

    ↓
  Deployment complete

RUNTIME FLOW:
  User visits site
    ↓
  HTML page loads with <script src="/js/env.js"></script>
    ↓
  window.ENV = { SUPABASE_URL: "...", SUPABASE_ANON_KEY: "..." }
    ↓
  Other scripts use window.ENV to connect to Supabase
    ↓
  Data loads from Supabase and displays
